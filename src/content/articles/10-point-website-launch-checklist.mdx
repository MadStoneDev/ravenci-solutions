Having a new website built for your business is hard work and when it's finally time to go live, it can be very exciting! How can you know if your website is ready to be published? Here's an extensive, printable website launch checklist. Don't go live without checking these!

### 1. Content Review & Proofreading

- [ ] All text is spell-checked and grammatically correct
- [ ] Contact information is accurate (phone, email, address)
- [ ] Business hours and locations are up-to-date
- [ ] All placeholder text (Lorem Ipsum) has been replaced
- [ ] Company name, tagline, and key messaging are consistent (this includes spelling)
- [ ] All images have descriptive alt text for accessibility

**Why it matters:** Nothing is worse for credibility than avoidable typos or incorrect contact information.

### 2. Mobile Responsiveness Test

- [ ] Website displays correctly on phones (iPhone & Android)
- [ ] All buttons and links are easily clickable on mobile
- [ ] Buttons and links do not depend on mouse hovering for functionality on mobile
- [ ] Text is readable without zooming
- [ ] Navigation menu works on mobile devices
- [ ] Contact forms function properly on all devices
- [ ] Images scale appropriately and load quickly

**Test on:** Safari on iPhone and iPad, Firefox and Chrome on Android and iOS

### 3. Page Loading Speed

- [ ] Homepage loads in under 3 seconds
- [ ] All interior pages load quickly
- [ ] Images are optimised and compressed (Hero/banner images: under 150kb, content images: under 80kb, thumbnails: under 30kb)
- [ ] No broken plugins or scripts causing delays
- [ ] Test your website from different internet connections
- [ ] Core Web Vitals scores are in "Good" range

**Tools:** Google PageSpeed Insights, GTmetrix, or similar speed testing tools

### 4. Contact & Lead Generation Setup

- [ ] Contact forms submit successfully and send notifications
- [ ] Auto-reply emails are working and professional
- [ ] Phone numbers are clickable on mobile devices
- [ ] Google Maps integration is accurate
- [ ] All inquiry methods are tested (form, phone, email)
- [ ] Privacy policy links are included on forms

**Test:** Submit actual test inquiries from different devices

### 5. Search Engine Optimisation (SEO) Basics

- [ ] Page titles are unique and descriptive for each page
- [ ] Meta descriptions are written for all pages
- [ ] Google Analytics is installed and working
- [ ] Google Search Console is set up
- [ ] Bing Webmasters Tools is set up
- [ ] XML sitemap is created and submitted to Google and Bing
- [ ] Local business schema markup is implemented (if applicable)

**Bonus:** Some of these will require help from your developer. Don't be afraid to ask them.

### 6. Cross-Browser Compatibility

- [ ] Website displays correctly in Chrome
- [ ] Website displays correctly in Safari
- [ ] Website displays correctly in Firefox
- [ ] Website displays correctly in Edge
- [ ] All interactive elements work in each browser
- [ ] No broken layouts or missing elements

**Don't forget:** Test on both Windows and Mac if possible (your developer should be able to do this)

### 7. Security & Technical Setup

- [ ] SSL certificate is installed and works (https:// not http://)
- [ ] All plugins/software are updated to latest versions
- [ ] Unnecessary plugins or features are removed
- [ ] Default admin passwords have been changed
- [ ] 404 error page is custom and helpful
- [ ] Website backups are configured and tested

**Security check:** Run basic security and malware scans to identify vulnerabilities from the get-go

### 8. Legal & Compliance

- [ ] Privacy policy is published and up-to-date
- [ ] Terms of service are included (if applicable)
- [ ] Copyright notices are in place
- [ ] Cookie consent is implemented (if required)
- [ ] Accessibility standards are met (WCAG basics)
- [ ] All stock photos are properly licensed

**Australia-specific:** Make sure that your website complies with Australian Privacy Principles

### 9. Social Media & Online Presence

- [ ] Social media links are working and current
- [ ] Social media profiles link back to new website (if applicable)
- [ ] Google My Business is updated with new website URL (if applicable)
- [ ] Online directory listings are updated (if required)
- [ ] Email signatures include new website URL (if required)

**Update everywhere:** If you are updating your website url, update it on Yellow Pages, industry directories, supplier portals, etc.

### 10. Final Testing & Launch Preparation

- [ ] All internal links are working correctly
- [ ] External links open in new windows
- [ ] Email addresses are working and monitored
- [ ] Website displays your business in the best light
- [ ] You're proud to send customers to this website
- [ ] Domain is pointing to the correct hosting location

**Launch day:** Monitor your site for any issues and have support contact ready

### Bonus Checklist Items (The "Nice to Haves"):

- [ ] Favicon is uploaded and displays correctly
- [ ] Website loads smoothly and quickly
- [ ] Newsletter signup is functional (if applicable)
- [ ] Online booking system is tested (if applicable)
- [ ] Live chat is working (if implemented)
- [ ] Video content plays without issues
